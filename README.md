# ⚔ Battle Simulator - Question Based Application

# Project Description

Battle Simulator is a Python Tkinter educational game that combines learning and entertainment through a battle-style quiz system. Players answer programming and computer-related questions to defeat robot enemies and progress through different difficulty levels.
The game was developed to make learning more interactive, engaging, and enjoyable for students interested in programming and technology.

# Objectives
- To create an interactive educational application
- To improve programming knowledge through quizzes
- To apply Object-Oriented Programming concepts
- To implement the SOLID Principles in software development
- To provide an enjoyable learning experience using gamification

# SDG Supported

This project supports:
#SDG 4 — Quality Education
The application promotes quality education by providing an engaging platform where students can learn programming and computer concepts interactively.

# 🎮 Features

✅ 100 Unique Questions  
✅ 5 Difficulty Levels  
✅ No Repeated Questions  
✅ Randomized Multiple Choice Answers  
✅ Health and Score System  
✅ Robot Battle Gameplay  
✅ Auto Next Question  
✅ Next Level Feature  
✅ Restart Game Feature  
✅ Try Again Feature  
✅ Modern GUI using Tkinter  

# Difficulty Levels
1. Easy  
2. Medium  
3. Hard  
4. Difficult  
5. Super Hard  
Each level contains 20 unique questions.

# Technologies Used
- Python
- Tkinter GUI
- Object-Oriented Programming (OOP)

# Installation Guide

# Step 1 — Install Python
Download Python from:
https://www.python.org/downloads/

# Step 2 — Download the Project
Clone the repository:

```
git clone https://github.com/yourusername/battle-simulator.git
```
Or download the ZIP file manually.

# Step 3 — Run the Program
Open terminal or command prompt:
```
python battle_simulator.py
```
